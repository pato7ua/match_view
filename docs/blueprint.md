# **App Name**: MatchView

## Core Features:

- Interactive Pitch Map: Display an interactive football pitch map with player positions using dummy GPS data.
- Player Orientation: Show player body orientation on the map using arrows based on BNO055 data.
- Player List Sidebar: A scrollable list of players with their name, speed, and battery level, which gets updated based on the dummy JSON data.
- Player Highlighting: Highlight a selected player on the pitch map when clicked in the player list.
- Match Controls: Implement start/pause, record session, and export data control buttons.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to evoke a sense of professionalism and technology.
- Background color: Very light blue (#E8EAF6), providing a soft, unobtrusive backdrop.
- Accent color: Electric Purple (#7C4DFF), used to highlight interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif font providing a modern look.
- Use minimalist icons that are easily understandable to represent player stats and controls.
- Implement a responsive layout with the pitch map in the center and the player list in a sidebar.
- Subtle animations for hover effects and data updates to enhance the user experience, aligning with the 'liquid glass' aesthetic, with elements that subtly change on hover, indicating interactivity.
